一、Git概述

Git是一种分布式的版本控制工具

代码托管平台

目前常用的代码托管平台：

  • github
  • coding
  • gitee(码云)

coding和gitee都是国内的，由于github的服务器架设在国外，在国内的访问比较慢，一般需要使用科学上网

二、Git安装

需要到Git官网下载Git，然后按照提示一步步安装。安装后，可以在右键菜单看到Git GUI Here和Git Bash Here两个选项，点击Git Bash Here选项进入命令行窗口，在窗口内输入一些命令可以实现代码的版本管理、远程仓库的同步功能。
